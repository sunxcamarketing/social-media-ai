THEMEN-SPEZIFIZITÄT (STRIKT):

Jeder Titel MUSS mindestens einen dieser vier SPEZIFITÄTS-ANKER enthalten:

1. **KONKRETE ZAHL** — ein Geld-Betrag, Prozent, Zeitraum, Anzahl (2%, 60.000€, 7 Monate, 4 Uhr morgens)
2. **BENANNTER GEGNER/TOOL/PERSON** — ein konkretes Objekt das man googlen könnte (Stop-Loss bei Crypto.com, TradingView-Indikator XY, Ray Dalio's Fehler)
3. **NICHT-OFFENSICHTLICHE THESE** — eine Aussage die der Zuschauer bisher nicht wusste oder bei der er widersprechen will ("Der teuerste Trading-Fehler ist nicht XY sondern…")
4. **SZENE STATT ABSTRAKTION** — ein konkreter Moment/Ort/Dialog ("Das Gespräch mit meinem Steuerberater bei dem ich realisierte…")

Ein Titel ohne mindestens EINEN dieser Anker ist automatisch ABGELEHNT.

# BAD → GOOD (Beispiele aus dem Trading/Freedom-Space)

❌ "Warum ein 9-to-5 dich niemals frei macht"
→ generisch, recycled, 0 Anker.
✅ "Der 62€-pro-Tag-Trick, warum dein Chef dich absichtlich beschäftigt hält"
→ Zahl + nicht-offensichtliche These.

❌ "Dieser Stop-Loss-Fehler kostet dich jeden Trade"
→ vage, Clickbait-Lücke ohne Payoff-Signal.
✅ "Warum ein 2%-Stop-Loss mathematisch der teuerste Fehler im Trading ist"
→ Zahl + These + fachlicher Begriff.

❌ "So erkennst du in 3 Schritten einen Fake-Guru"
→ "5 Tipps"-Format, austauschbar.
✅ "Der eine Satz in jeder Trading-Werbung der beweist: Fake-Guru"
→ benannter Anker + nicht-offensichtliche These.

❌ "Von meinem ersten 500€-Konto zu 60.000€ an einem Tag"
→ klingt fake, kein Kontext.
✅ "500€ → 60k in 7 Monaten → 0€ in 3 Tagen. Das hab ich gelernt."
→ Zahlen + Story-Arc + Lesson.

❌ "Der Morgen, an dem ich nie wieder pendeln musste"
→ soft, kein Konflikt.
✅ "Der 4:47-Uhr-Call mit meiner Schwester, der mir klar machte: Freiheit fühlt sich anders an als gedacht"
→ Szene + Konflikt + Counter-Expectation.

# REJECT-LISTE (automatisch ablehnen wenn Titel matched)

Diese Titel-Muster sind verboten — sie klingen wie 2020-Motivations-Bro-Content:
- "Warum ein/die/das [abstrakter Begriff] dich [verb]" ohne Zahl oder benannten Anker
- "So erkennst du in X Schritten…" / "X Zeichen dass…" ohne besonderen Dreh
- "Von [Zahl]€ zu [Zahl]€ an einem Tag/in einer Woche" ohne konkreten Zeitrahmen + Mechanismus
- "Der Moment/Der Tag/Der Morgen an dem…" ohne konkrete Szene (Uhrzeit, Ort, Personen, Dialog)
- "5 Tipps/Dinge/Fehler die…" (Listicle-Format außer der Titel hat einen Kontra-Twist)
- Alles mit "niemals", "immer", "alle", "jeder" als absolute Behauptung ohne Zahl-Beleg

# POLITIK STATT PREDIGT

Jeder Titel sollte einen GEGNER haben. Nicht "dich motivieren" sondern "gegen etwas stehen":
- Gegen ein falsches Glaubensmuster
- Gegen einen Konkurrenten / Guru / eine Methode
- Gegen die eigene alte Version
- Gegen eine Industrie-Lüge

Titel die niemandem auf die Füße treten sind Titel die niemanden bewegen.
